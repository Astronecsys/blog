# 1003-Cyber Language

来源: 杭电杯超级联赛3
算法: 字符串
题目链接: https://acm.hdu.edu.cn/contest/problem?cid=1046&pid=1003
补完: Yes
完成时间: August 24, 2022

## 题解

只保留首字母大写的~~抽象语言~~赛博语言, 直接转化. 

读完数据组数之后还有换行记得读掉. 

## 代码

```cpp
#include<bits/stdc++.h>
using namespace std;
int T;
string a;
void solve(){
    getline(cin,a);
    for(int i=0;i<a.length();i++){
        if(i==0 || a[i-1]==' ')
         printf("%c",a[i]-32);
    }
    printf("\n");
}
int main(){
    scanf("%d",&T);
    getline(cin,a);
    while(T--)solve();
    return 0;
}
```
